## client.py

客户端程序 具有3个函数：

1.SettPortIp（）：
	此函数主要进行Windows客户端与Linux的网络连接，采用 socket-python 中的connect方法进行服务端IP地址以及端口的连接
	其中 IP 地址类型为str类型， 服务器端口为 int 类型 ， 该函数并未直接进行传参，而是直接采用 try 方法直接进行直连， 若连接
	成功则返回 “conntect success”，连接失败则退出程序，其中设定了连接最大时间，若抄出最大时间，则返回Time_out;

2. SendData(path):
	该函数用来进行图片的传输，传递一个path参数， 即图片路径（str类型）， 该思路是将图片转为128sq格式打包，并以二进制数据
	格式进行传输，在服务端进行数据的读入， 服务端读入成功后给予客户端一个反馈，并返回一个列表给客户端

3.Monitor(path_monitor):
	该函数主要负责对目标文件夹的监控，程序默认采用的是当目标文件夹（监控路径自行设定）中添加新图片即可进行传输,可根据实际需要对其更改。


## server.py

服务端程序仅一个函数，用来接受图片，并将图片格式进行还原，默认保存在程序根目录下。
